# Geospatial Analysis Portfolio (QGIS)

This repository contains a collection of geospatial analysis projects developed using QGIS, focused on spatial data processing, map creation, and analytical workflows.  

The goal of this portfolio is to demonstrate practical skills in handling geographic data and producing accurate, well-structured outputs suitable for real-world applications and AI training tasks.

---

## 🛠️ Tools & Technologies

- QGIS  
- Python (basic data processing)  
- SQL (data querying)  
- Geospatial data formats (Shapefile, GeoJSON, CSV, Raster data)  

---

## 📂 Projects

### 1. Land Use Mapping Project
**Description:**  
This project involves the creation of a land use classification map using spatial datasets. The objective was to organize, analyze, and visualize geographic data to identify different land use categories.

**Key Tasks:**
- Imported and processed vector datasets  
- Applied Coordinate Reference Systems (CRS) and reprojection  
- Performed attribute filtering and classification  
- Designed map layout with legend, scale, and labels  

**Output:**
- Final map showing categorized land use  
- Clean and structured visual presentation  

---

### 2. Spatial Buffer Analysis
**Description:**  
This project focuses on analyzing spatial relationships using buffer zones around key features such as roads or points of interest.

**Key Tasks:**
- Created buffer zones at defined distances  
- Analyzed spatial relationships between features  
- Used overlay tools to extract relevant data  
- Validated outputs for accuracy  

**Output:**
- Map showing buffer zones and affected areas  
- Spatial insights derived from analysis  

---

### 3. Data Cleaning & Processing
**Description:**  
This project demonstrates data preparation techniques for structured datasets used in geospatial workflows.

**Key Tasks:**
- Cleaned and formatted datasets using Python/SQL  
- Removed inconsistencies and handled missing values  
- Prepared data for integration into GIS tools  

**Output:**
- Clean datasets ready for analysis  
- Improved data accuracy and usability
